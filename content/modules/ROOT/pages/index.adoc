= Agentic AI Super Lab

== Lab Overview

This lab illustrates how agentic AI can be applied to help solve a real-world business problem. You will explore basic agentic AI capabilities and deploy an end-to-end agentic AI solution.

By the end of this workshop, participants will have:

* ✅ A fully functional AI agent that can autonomously handle CI/CD failures
* ✅ Understanding of agent development lifecycle and deployment
* ✅ Experience with observability and evaluation of AI systems
* ✅ Practical knowledge of integrating AI agents into real-world workflows

== Workflow

1. Introduction and Overview
2. Using Llama Stack
3. Configure and Deploy your Agent
4. Set up Safety Guardrails
5. Agentic System Observability
